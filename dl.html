下载网站部署文件

点下面两个按钮，分别把文件存到手机「下载/Downloads」里。

⬇️ 下载 index.html ⬇️ 下载 latest.json
⚠️ 最容易踩的坑：文件名变成 (1)
手机下载时，如果「下载」里已有一个同名文件，会自动改名成 index(1).html。GitHub Pages 只认 index.html 这个名字，如果传上去的是 index(1).html，旧网站不会变！
上传前请先在手机里把文件名改回 index.html（长按文件 → 重命名 → 删掉 (1)）。
下载后怎么上传到 GitHub（只需上传 index.html）：
1. 打开手机浏览器，进你的仓库（仓库名以你浏览器地址栏里实际的为准）；
2. 先删除仓库里误传的 index(1).html：点它 → 右上角 ⋯ → Delete file → Commit；
3. 在手机「下载」里把文件重命名为 index.html（删掉 (1)）；
4. 回 GitHub 仓库，点右上角 “Add file” → “Upload files”；
5. 把重命名后的 index.html 传上去（会替换旧的）；
6. 拉到最下面点绿色 “Commit changes”；
7. 等 1–2 分钟，访问 https://pma8568.github.io/Nasdaq100-PE/ 即可。

说明：网站每天会自动抓取最新数据，你不需要每天更新。只有在我改了网站功能（比如这次优化图表）时，才需要重新上传一次 index.html。
